# 10.2: Nonconformity and corrective action

When nonconformity occurs: react (including mitigating environmental impacts), evaluate need for corrective action, implement action, review effectiveness, make changes to EMS if needed.

> Use the Corrective Actions module for formal CAPA tracking.
