# PI1.5: Storage and protection of inputs and outputs

## Purpose

The entity stores inputs, items in processing, and outputs completely, accurately, and
timely in accordance with system specifications to meet the entity's objectives.

## Description

The organization implements controls to ensure that stored data (inputs, intermediate
processing data, and outputs) maintains its integrity and is protected from unauthorized
modification or destruction. Data retention policies define how long data is stored.

## Key Considerations

- Data storage controls maintain integrity of stored data
- Data retention and disposal policies are defined
- Stored data is protected from unauthorized modification
- Backup and recovery controls protect stored data
- Storage systems are monitored for integrity
