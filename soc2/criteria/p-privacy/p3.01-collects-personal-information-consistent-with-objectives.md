# P3.1: Collection of personal information

## Purpose

Personal information is collected consistent with the entity's objectives related
to privacy.

## Description

The organization limits the collection of personal information to what is necessary
for the identified purposes. Collection methods are lawful, fair, and transparent.
The types of personal information collected are documented and aligned with the
stated purposes in the privacy notice.

## Key Considerations

- Data minimization principles are applied
- Collection is limited to stated purposes
- Collection methods are lawful and transparent
- Types of personal information collected are documented
- Special categories of data have additional collection controls
