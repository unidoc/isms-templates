# P5.1: Data subject access

## Purpose

The entity grants identified and authenticated data subjects the ability to access their
stored personal information for review and, upon request, provides physical or electronic
copies of that information.

## Description

The organization implements procedures for data subjects to access their personal information,
request corrections, and obtain copies. Access requests are verified, processed within
regulatory timeframes, and documented.

## Key Considerations

- Data subject access request (DSAR) process is defined
- Identity verification is performed before granting access
- Requests are processed within regulatory timeframes
- Data subjects can request corrections to inaccurate data
- Access request logs are maintained
