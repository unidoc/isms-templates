# CC6.8: Prevention and detection of unauthorized or malicious software

## Purpose

The entity implements controls to prevent or detect and act upon the introduction of
unauthorized or malicious software to meet the entity's objectives.

## Description

The organization deploys anti-malware controls, application whitelisting, software
restriction policies, and endpoint detection and response tools. Processes exist to
ensure that only authorized software is installed and that malicious software is
detected and removed promptly.

## Key Considerations

- Anti-malware software is deployed on all endpoints and servers
- Application whitelisting or software restriction policies are in place
- Endpoint detection and response (EDR) provides real-time threat detection
- Software installation is restricted to authorized personnel
- Malware definitions and detection signatures are updated regularly
