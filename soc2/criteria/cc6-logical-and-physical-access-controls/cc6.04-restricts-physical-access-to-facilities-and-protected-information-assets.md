# CC6.4: Physical access restrictions

## Purpose

The entity restricts physical access to facilities and protected information assets
(e.g., data centres, backup media) to authorized personnel.

## Description

The organization implements physical security controls that restrict access to facilities,
server rooms, data centres, and other areas containing protected information assets. Physical
access is controlled through mechanisms such as badge readers, biometric controls, security
guards, and visitor management procedures.

## Key Considerations

- Physical security perimeters are defined
- Access control mechanisms are implemented (badge readers, biometrics)
- Visitor management procedures are in place
- Physical access logs are maintained and reviewed
- Data centre access is restricted and monitored
