# CC6.5: Disposal and discontinuation of asset protections

## Purpose

The entity discontinues logical and physical protections over physical assets only after
the ability to read or recover data and software from those assets has been diminished and
is no longer required.

## Description

The organization has processes for securely decommissioning and disposing of hardware and
media that contained sensitive data. Data destruction methods ensure that data cannot be
recovered, and disposal is documented.

## Key Considerations

- Data destruction policy defines approved methods (degaussing, shredding, wiping)
- Asset disposal is documented with certificates of destruction
- Disposal methods are appropriate to data classification
- Third-party disposal vendors are vetted
- Cloud resource decommissioning procedures exist
