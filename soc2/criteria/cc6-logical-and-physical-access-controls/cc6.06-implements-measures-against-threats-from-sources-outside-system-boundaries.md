# CC6.6: Security against threats from outside system boundaries

## Purpose

The entity implements logical access security measures to protect against threats from
sources outside its system boundaries.

## Description

The organization deploys boundary protection controls including firewalls, intrusion
detection/prevention systems, web application firewalls, DDoS protection, and secure
communication protocols. Network traffic is monitored and filtered at system boundaries.

## Key Considerations

- Firewalls and network access control lists are configured
- Intrusion detection/prevention systems are deployed
- Web application firewalls protect public-facing services
- DDoS mitigation measures are in place
- Secure protocols (TLS, SSH) are used for external communications
