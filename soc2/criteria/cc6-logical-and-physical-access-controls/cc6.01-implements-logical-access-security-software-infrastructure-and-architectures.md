# CC6.1: Logical access security software, infrastructure, and architectures

## Purpose

The entity implements logical access security software, infrastructure, and architectures
over protected information assets to protect them from security events.

## Description

The organization deploys security controls including identity and access management systems,
network security controls (firewalls, IDS/IPS), endpoint protection, and encryption. The
security architecture is designed to protect information assets based on their classification
and the threat landscape.

## Key Considerations

- Identity and access management (IAM) system is implemented
- Network segmentation and firewall rules are defined
- Endpoint detection and response (EDR) is deployed
- Encryption is used for data at rest and in transit
- Security architecture is documented and reviewed
