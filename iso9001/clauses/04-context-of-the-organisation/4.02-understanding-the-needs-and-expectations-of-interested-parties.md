# 4.2: Interested parties

## Purpose

Determine interested parties relevant to the QMS and their relevant requirements.

| Party | Requirements | How addressed |
|-------|-------------|---------------|
| Customers | Product/service quality, delivery, support | |
| Employees | Working conditions, competence development | |
| Suppliers | Fair terms, clear specifications | |
| Regulators | Compliance with applicable laws | |
| Shareholders | Business performance, reputation | |
